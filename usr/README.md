# Modern Flutter Website

A beautiful, fully responsive website landing page built with Flutter. 

## Features
- **Responsive Navigation**: Adaptive header with desktop links and a mobile hamburger menu.
- **Hero Section**: High-impact section with clear calls to action.
- **Features Grid**: Responsive feature highlights that scale automatically from mobile single-column to desktop multi-column layouts.
- **Cross-Platform**: Natively compiled for iOS, Android, Web, Mac, Windows, and Linux.

## Setup & Run
Make sure you have Flutter installed. Then run:
```bash
flutter run -d web
```

## Tech Stack
- Framework: Flutter
- UI Components: Material 3
